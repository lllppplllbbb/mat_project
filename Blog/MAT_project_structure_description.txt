MAT_project 项目各文件作用说明
================================

[根目录文件]
------------
- train.py                ：训练入口脚本，加载数据、模型和损失函数等，并开始训练过程。
- generate_image.py       ：推理脚本，使用训练好的模型生成修复图像。
- dataset_tool.py         ：数据处理工具，例如生成掩码或转换数据格式。
- legacy.py               ：旧代码兼容模块，可能用于加载旧模型等。
- README.md               ：项目介绍。
- LICENSE                 ：开源协议。
- requirements.txt        ：项目依赖库。
- .gitignore              ：Git 忽略文件配置。

[datasets/ 数据集模块]
-----------------------
- dataset_512.py          ：加载 512×512 尺寸的训练数据集。
- dataset_512_val.py      ：加载验证集。
- mask_generator_512.py   ：生成用于训练的大尺寸掩码。
- mask_generator_512_small.py：生成尺寸较小的掩码。

[dnnlib/ 工具库]
----------------
- util.py                 ：通用工具函数，如路径管理、日志等。
- __init__.py             ：使 dnnlib 成为 Python 包。

[evaluatoin/ 模型评估（疑似拼写错误，应为 evaluation）]
-----------------------------------------------------
- cal_fid_pids_uids.py    ：计算 FID 及身份/用户指标。
- cal_lpips.py            ：计算 LPIPS 指标。
- cal_psnr_ssim_l1.py     ：计算 PSNR、SSIM、L1 等指标。

[losses/ 损失函数模块]
-----------------------
- loss.py                 ：主损失函数封装。
- pcp.py                  ：结构感知损失函数。
- vggNet.py               ：VGG 感知损失网络。

[metrics/ 指标工具集]
----------------------
- frechet_inception_distance.py：FID 指标。
- inception_score.py      ：IS 指标。
- kernel_inception_distance.py：KID 指标。
- metric_main.py          ：指标计算主入口。
- metric_utils.py         ：指标计算辅助工具。
- psnr_ssim_l1.py         ：PSNR、SSIM、L1指标。
- precision_recall.py     ：用于计算精确度和召回率等。
- perceptual_path_length.py：感知路径长度指标。
- __init__.py             ：使 metrics 成为 Python 包。

[networks/ 模型结构]
---------------------
- basic_module.py         ：定义基本模块如卷积、残差块等。
- mat.py                  ：MAT 主模型定义，含 Transformer 结构等。

[torch_utils/ PyTorch 工具库]
-----------------------------
- custom_ops.py           ：自定义 PyTorch 操作。
- misc.py                 ：杂项工具函数。
- persistence.py          ：模型保存/恢复工具。
- training_stats.py       ：训练过程统计工具。
- ops/                    ：高效 CUDA 实现的操作（如 upfirdn2d, bias_act 等）。

[training/ 训练逻辑]
---------------------
- augment.py              ：训练数据增强操作。
- training_loop.py        ：训练主循环，包括前向、反向传播和优化。

[tests_add/ 自定义测试]
------------------------
- torch_test.py           ：作者自定义的 PyTorch 模块测试脚本。

[test_sets/ 测试数据集]
------------------------
- CelebA-HQ/              ：CelebA-HQ 测试图像和掩码。
- Places/                 ：Places 测试图像和掩码。
